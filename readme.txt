Обработка отрицательных чисел: Для правильной работы с отрицательными числами тебе нужно обрабатывать ситуации, 
когда пользователь вводит минус перед первым числом. Убедись, что при нажатии на - программа не воспринимает это сразу как оператор.
Можно добавить логику, которая проверяет, является ли первое число отрицательным или уже введён оператор.
